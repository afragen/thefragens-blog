Playing with the OS X Upstreaming via a Folder Action. It's not all sunshine and roses. It works great if you happen to be adding or deleting a file from the www folder. However, if it's another folder or a file in a subfolder of the www folder then no joy. Even when making a new blog post I don't think any files inside the www folder are added or removed and therefore the folder action isn't called -- bummer. Perhaps some combination of increasing the upstreaming time interval and use of the folder action -- but then why bother with the folder action at all... I'll let it run and see if this post gets upstreamed.
