So @tommcfarlin has finally made his [Mayer](https://shop.pressware.co/mayer/) theme available for WordPress.org sites. I've been following Tom's progress with Mayer and I had to make it my first purchased WordPress theme. I really appreciate Tom's (and WordPress') core philosophy, "decisions, not options". It allows for much more efficient code. I'm really looking forward to how Mayer evolves.
